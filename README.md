## 技術スタック
| 分野 | 使用技術 |
| ---- | ---- |
| フロントエンド | Vue(TypeScript), Vue Router, Pinia(Global Store), Element-Plus(UI Library) |
| バックエンド | Gin(Go), Air(Hot Reload), golang-migrate |
| データベース | MySQL |
| Webサーバー | Nginx(Reverse Proxy) |
| その他 | Docker, SSL |
